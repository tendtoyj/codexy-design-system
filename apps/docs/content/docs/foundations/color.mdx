PDS의 색은 세 층으로 구성된다.

1. **Primitive** — 14 hue × 14단계의 atomic 팔레트. 직접 소비 금지, semantic 계층이 참조.
2. **Semantic** — 의미 단위 토큰. 컴포넌트는 여기만 참조한다. 브랜드 값 교체는 primitive에서 한 번에 끝나므로 semantic 이름은 불변.
3. **Chat** — AI/Agent UI 전용. 별도 AI 카테고리 대신 semantic alias + 전용 glow 한 종.

> 현재 `blue.50 = #0066FF`는 The Pluto 브랜드 확정 전 잠정값. 확정 시 primitive 값만 교체하며, semantic 이름은 불변.

## Primitive 팔레트

Hue 14종 × 단계(주로 10/20/.../90/95/99). 이름 규칙: `--pds-color-{hue}-{step}`.

## Semantic 토큰

컴포넌트가 참조하는 실사용 계층. 값 대부분은 primitive의 참조, 일부는 `color-mix()`로 알파를 섞는다.

## 다크 모드

현재는 라이트 값만 정의되어 있다. 다크 값은 추후 `[data-theme="dark"]` 블록으로 추가될 예정이다.
